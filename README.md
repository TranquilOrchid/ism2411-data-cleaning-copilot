# ISM2411 Data Cleaning Project with GitHub Copilot

This project demonstrates a Python-based data cleaning workflow on a messy sales dataset. The goal is to show proficiency in cleaning real-world CSV data, handling missing and invalid values, normalizing text, and preparing clean, recruiter-ready output.

## Project Structure

ism2411-data-cleaning-copilot/
├── data/
│   ├── raw/
│   │   └── sales_data_raw.csv
│   └── processed/
│       └── sales_data_clean.csv
├── src/
│   └── data_cleaning.py
├── README.md
└── reflection.md

- `data/raw/sales_data_raw.csv` – Original messy dataset.  
- `data/processed/sales_data_clean.csv` – Cleaned CSV generated by the script.  
- `src/data_cleaning.py` – Python script implementing the cleaning pipeline.  
- `reflection.md` – Reflection on using GitHub Copilot and learning outcomes.  

## How to Run

1. Make sure you have Python 3 installed.  
2. Create and activate a virtual environment (optional but recommended):

```bash
python3 -m venv .venv
source .venv/bin/activate  # macOS/Linux
.venv\Scripts\activate     # Windows

3. Install required packages:

pip install pandas

4. Run the cleaning script:

python src/data_cleaning.py

5. The cleaned CSV will be saved to data/processed/sales_data_clean.csv
